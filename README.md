# Calculator-App

A modern, responsive calculator built with HTML, CSS, and JavaScript.  
It supports basic arithmetic operations, keyboard input, dark/light mode, and a clean user-friendly interface.

##  Features

 - Basic operations (Addition, Subtraction, Multiplication, Division)  
 - Keyboard support  
 - Dark / Light theme toggle  
 - Responsive design (works on mobile & desktop)  
 - Percentage and sign toggle (+/âˆ’)  
 - Error handling (division by zero)  
 - Formatted numbers with commas  
 - Local storage theme saving  

##  Built With

- HTML5  
- CSS3 (Responsive UI & Themes)  
- JavaScript 

## How It Works

The calculator is built using a JavaScript class that:

- Handles user input
- Manages operations
- Updates the display dynamically
- Supports both button clicks and keyboard input
- Saves theme preference in local storage

##  Responsive Design

The layout adjusts smoothly for:

- Mobile phones  
- Tablets  
- Desktop screens
